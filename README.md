# TMF Sandbox - IT Fundamentals

This repository serves as the central sandbox environment for the TMF Scholar Development Assignment in **IT Fundamentals**.

## Purpose
- Store documentation, system maps, runbooks, scenarios, and general learning materials.
- Support structured training without affecting any production systems.
- Act as a reference for future TMF scholars.

## Boundaries (Sandbox Only)
- This is a **training sandbox** only.
- No real personal data, company data, or sensitive information may be stored here.
- All experiments and "broken" scenarios must be clearly documented.
- Changes must be committed with clear messages.

## Sandbox Rules
- Never push real credentials, API keys, or production configurations.
- All work here is for learning and simulation purposes.
- Follow the TMF Sandbox folder structure (mirrored in documentation).

## Folder Structure Reference
See the main TMF-SANDBOX in Zoho WorkDrive for the complete organization.

## How to Use
1. Clone the repo: `git clone https://github.com/yourusername/tmf-sandbox-it-fundamentals.git`
2. Document all activities in Markdown.
3. Keep everything organized and reusable.

**Note**: This repository is deliberately separated from any production or live projects.
